Examples of Independence in the Simply Typed Lambda Calculus in Abella



This webpage contains some supporting material for the paper:

“ A Proof-theoretic Characterization of Independence in Type Theory” by Yuting Wang and Kaustuv Chaudhuri

You may browse the following development on-line. To run them yourself, you will need Abella in the master branch containing the most recent bug fixes, which is available from the main Abella web-page.

Independence between Types and Terms in Simply Typed Lambda Calculus

  • stlc contains strengthening lemmas that show types in simply typed lambda calculus (STLC) do not depend on terms a) in an empty context (the example in Section 5.1), and b) in a context containing terms.

Additional Examples of Independence

  • str contains the proof for the example in Appendix B.1
  • str1 contains the proof for the example in Appendix B.2. By adding a clause that introduces the dependence between q and r, the independence of q and r can no longer be proved.
