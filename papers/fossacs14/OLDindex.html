Bisimulation up-to techniques in Abella



NOTE: this page is currently being developed. It should be ready in a few more days. (17 Oct 2013).


This webpage contains some supporting material for the paper:

Formalization of the Bisimulation-Up-To Technique and its Meta Theory. Kaustuv Chaudhuri, Matteo Cimini, and Dale Miller,

which was submitted to FoSSaCS 2014.

The formal specifications in this paper are written with a slight extension to Abella where some higher-order quantification is allowed. While our use of such higher-order quantification could easily be replaced with only first-order quantification, the readability of the specifications would be decreased. In order to get Abella to treat this particular notion of higher-order quantification in coinductive proofs, the lax_coinduction branch of Abella is needed (for more information, see this). This is available on github as the following zip file.


Complete formalizations

  • Up to techniques for CCS: ccs.thm. This file contains the specification of CCS, definition of various techniques: up-to bisimilarity, up-to transitivity, up-to context, and up-to bisimularity and context. Also, the abstract definitions for soundness theorems and the soundness of those technique for bisimilarity for CCS. The files also contains several examples illustrating how to use these results.
  • Up to techniques for π-calculus: pic.thm.
    Important Remark: At page 8 and 10 of the paper, we mention in passing that we consider late bisimilarity. It is a text mishappening, the induced bisimilarity is open bisimilarity, as proved in Proof search specifications of bisimulation and modal logics for the calculus by Alwen Tiu, Dale Miller, ACM Transactions on Computational Logic (TOCL) 2010, and as stated in Section 5 where bisimilarity for π-calculus is given.
    This file contains the specification of π-calculus, the definition of various techniques: up-to bisimilarity, up-to transitivity, up-to context, and up-to bisimularity and context. Also, the abstract definitions for soundness theorems and the soundness of those technique for bisimilarity for CCS. The files also contains several examples illustrating how to use these results.
  • Up to techniques for weak bisimilarity, CCS: ccs_weak.thm and π-calculus: pic_weak.thm.
    Important Remark: At page 8 of the paper, we mention quickly that we consider weak bisimilarity. We have so far considered only the ordinary definition of weak bisimilarity (i.e. that basically plays strong bisimilarity on weak transitions. This definition does not pose problems with up-to techniques, and has a smooth theory of up-to techniques. However, this formulation is usually not useful in practice for proofs.

In Progress

  • Weak bisimulation for CCS and π-calculus, up-to techniques for the "problematic" weak bisimilarity.
  • Up-to techniques for Higher-Order calculi.
  • Formalization of the theory of compatible functions.
  • Enhancement of Model Checking by the use of Lemma. (As advocated in the paper Extracting Proofs from Tabled Proof Search by Dale Miller and Alwen Tiu, CPP 2013.) Bedwyr is a model checker based on recent advances in proof theory. Since its underlying logic is basically the same of the Abella theorem prover, our plan is to enable Bedwyr to make use of lemmas from Abella, available only when formally proved and that can therefore be trusted, in order to avoid the exhaustive search of states space and perform model checking in a smarter, yet sound, way.
