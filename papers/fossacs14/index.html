Formalized Meta-Theory of Bisimulation-Up-To Techniques in Abella



This webpage contains some supporting material for the paper:

Formalization of the Bisimulation-Up-To Technique and its Meta Theory by Kaustuv Chaudhuri, Matteo Cimini, and Dale Miller,

which was submitted to FoSSaCS 2014. The links below are to the HTML version of the Abella developments. The actual Abella sources are also linked therefrom, but the development can be browsed without running Abella yourself.

Note: the development below requires the version of Abella from Git because it relies on some recent bug-fixes. This version of Abella is available as a tarball or a zip from Github. These bugfixes will be part of the next release (2.1.x) of Abella.


Complete formalizations

  • Strong bisimilarity for CCS
    This file contains the specification of CCS, definitions of various techniques: up-to-bisimilarity, up-to-context, up-to-bisimularity-and-context, and a variant of up-to-transitivity. It also contains the proofs of soundness of these techniques with respect to bisimilarity for CCS.

    Several illustrative examples using these theorems are also provided.
  • Strong bisimilarity for the π-calculus
    An extension of the above development for the π-calculus.

    Also contains several updated examples that make use of communication.
  • Weak bisimilarity using the ordinary definition: CCS and the π-calculus
    These definitions play the strong bisimulation game on weak transitions.
  • Weak bisimilarity with a strong challenger: CCS and the π-calculus
    This is a variant of weak bisimilarity that is better suited for proofs. In the bisimulation game, the challenger plays strong transitions. This is the weak bisimilarity that has been extensively studied in the context of up-to-techniques.
