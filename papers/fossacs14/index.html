Some formalizations of bisimulation-up-to techniques in Abella


Below we place various files that contain formalizations (or partial formalizations) of techniques related to bisimulation-up-to for CCS and π-calculus.

Supporting material

K. Chaudhuri, D. Miller, and M. Cimini have written and submitted a paper titled Formalization of the Bisimulation-Up-To Technique and its Meta Theory. Below we list several files containing Abella formulations that support the formalization claims in that paper.

  • The file pic.thm contains a specification of one-step transitions, bisimulation, bisimulation up to bisimularity, bisimulation up to transitivity, bisimulation up to context, and bisimulation up to bisimularity and contexts. The soundness proofs for all these up-to-techniques are included as are several examples illustrating how to use these results.
