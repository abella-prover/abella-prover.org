A Two-Level Logic Approach to Reasoning about Typed Specification Languages



This webpage contains some supporting material for the paper:

A Two-Level Logic Approach to Reasoning about Typed Specification Languages by Mary Southern and Kaustuv Chaudhuri,

which was submitted to FSTTCS 2014.

We have fixed an error on page 8 of the paper.

The links below are to the HTML version of the Abella developments. The actual Abella sources are also linked therefrom, but the development can be browsed without running Abella yourself.

Note: the development below requires the version of Abella from Git. This version of Abella is available as a zip from Github.

    This file contains the specification of the simply typed lambda calcuclus and the typing judgement. It also contains a proof of type uniqueness for these terms.
